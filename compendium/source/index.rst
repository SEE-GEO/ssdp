Scientifc Software Development with Python
==========================================

Welcome to the course homepage for the course *Scientific Software Development with
Python* (SSDP). In addition to the lecture slides I will try to provide here a
technical compendium that describes how to use the tools presented in the course.

Course plan and project requirements
------------------------------------

The course plan describing course content and intended learning outcome as
well as a detailed plan for the project work can be downloaded from the following two links:

- `course_plan.pdf <https://github.com/SEE-MOF/ssdp/blob/main/course_plan/ssdp.pdf>`_
- `project_plan.pdf <https://github.com/SEE-MOF/ssdp/blob/main/course_plan/project_plan.pdf>`_

A detailed schedule for the lectures can be found under `lectures <lectures.html>`_.
